# Check Model Status

Retrieve the current training status of a machine learning model.

## Endpoint
**Method:** `GET`  
**Path:** `/v0/models/{model_name}/{model_version}/status`

---

## Headers

| Name         | Type   | Required | Description                          |
|--------------|--------|----------|--------------------------------------|
| `x-api-key`  | string | Yes      | Your API key for authentication.     |

---

## Parameters

### Path Parameters

| Name            | Type   | Required | Description                                              |
|------------------|--------|----------|----------------------------------------------------------|
| `model_name`     | string | Yes      | The name of the model whose status you want to check.    |
| `model_version`  | string | Yes      | The version ID of the model returned during model creation. |

---

## Responses

### 200 - Successful Response
**Description:** Indicates the current status of the model.  

**Example:**
```json
{
  "status": "completed",
  "result": {
    "code": "def predict(data): ...",
    "valid_metric": 0.87
  },
  "error": null
}
